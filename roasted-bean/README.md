# Roasted Bean Coffee – WordPress Theme

A custom WordPress theme built for a fictional coffee shop website.  
This theme was developed as part of a frontend + WordPress portfolio project.

## Features

- Custom WordPress theme
- Responsive layout
- Custom navigation menu
- Custom logo support
- Multi-page structure
- Theme configuration using `theme.json`
- Styled homepage sections
- Menu page layout
- Contact page layout
- Theme screenshot preview

## Pages Included

- Home
- Menu
- About
- Contact
